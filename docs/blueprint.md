# **App Name**: Oshadi Vidarshana

## Core Features:

- Homepage: Display Oshadi Vidarshana's introduction and a summary of his experience.
- About Me: Detailed information about Oshadi Vidarshana's background, skills, and expertise.
- Projects Portfolio: Showcase Oshadi Vidarshana's projects with descriptions, images, and links to live demos or GitHub repositories.
- Contact Form: Allow visitors to send messages directly to Oshadi Vidarshana.
- Blog: A place to share thoughts, ideas, tutorials, or project updates with the website audience. A CMS can also be used as an add-on

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A), conveys professionalism and trust.
- Background color: Very light gray (#F5F5F5), for a clean and modern look.
- Accent color: Teal (#34D399), for highlighting key elements.
- Body font: 'Inter', a sans-serif font for readability and modern feel.
- Headline font: 'Space Grotesk', a bold sans-serif for titles.
- Use a consistent set of line icons related to skills and projects.
- Clean, responsive layout with a focus on visual hierarchy and user experience.
- Subtle transitions and animations to enhance user engagement.
